<h1 align="center">Description</h1>

<p align="center">
Consider a number N with up to 1000 digits represented as a linked list such that every digit N represent a node in the list. You need to increment the number N in 1.
</p>

<br> <br>
## Input
List : 1->2->9->nullptr

## Output
List : 1->3->0->nullptr
