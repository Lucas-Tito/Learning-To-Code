    • \[\mathbf{T(n) = 4T(\lfloor \frac{n}{2} \rfloor) + cn}\]

    No método da árvore de recorrência precisamos analisar algumas constantes. Assim, tomando:

    a=4 b=2 f(n)=cn

    temos constantes que ditam respectivamente: as divisões dos nós, o tamanho dos nós e o custo de cada nó. Além disso, no método da árvore podemos ignorar o piso.



    Imagem apenas para meus estudos, avaliador favor ignorar.



    Podemos utilizar uma tabela para analisar melhor a árvore:

    \begin{array} {|r|r|}\hline \text{Nível} & \text{Tamanho do Nó} & \text{N Nós} & \text{Custo por Nó} & \text{Custo do Nível} \\ \hline 0 & \frac{n}{2⁰} & 4⁰ & c*\frac{n}{2⁰} & 4⁰*c*\frac{n}{2⁰} \\ \hline 1 & \frac{n}{2¹} & 4¹ & c*\frac{n}{2¹} & 4¹*c*\frac{n}{2¹} \\ \hline 2 & \frac{n}{2²} & 4² & c*\frac{n}{2²} & 4²*c*\frac{n}{2²} \\ \hline i & \frac{n}{2^{i}} & 4^{i} & c*\frac{n}{2^{i}} & 4^{i}*c*\frac{n}{2^{i}} \\ \hline log_2 (n) & \frac{n}{2^h}=1 & 4^{log_2 (n)} & \Theta(1) & n²\\ \hline \end{array}

    A nosso última linha da tabela corresponde ao último nível da árvore. Nesse momento o nosso tamanho do nó é dado por \( \ \ \mathbf{\frac{n}{2^{h}} \ = \ 1 \ \ } \) isso procede já que estamos igualando a nossa forma genérica descoberta na linha anterior da tabela a 1, que é quando chegamos nas folhas da árvores (que correspondem a \( \Theta(1) \)).



    Para determinar a altura do último nível podemos desenvolver o tamanho do nó da seguinte maneira:

    \[ \frac{n}{2^{h}} = 1 \ \ \ \ = \ \ \ \ n = 2^{h} \ \ \ \ = \ \ \ \ log_2 (n) = log_2 (2^{h}) \ \ \ \ = \ \ \ \ log_2 (n) = h * log_2 (2) \] \[ = \ \ \ \mathbf{log_2 (n) = h} \]

    Por fim, para determinar o custo total da recorrência devemos usar um somatório que vai do primeiro nível até o pneultimo nível da árvore. Além disso, no termo do somatório temos a soma do custo até o penúltimo nível mais o último nível da árvore, logo temos:

    \[ T(n) \ = \ \sum_{i=0}^{log_2 (n-1)} \ 4^{i} * c * \frac{n}{2^{i}} + n²\] \[ = \ \sum_{i=0}^{log_2 (n-1)} \ cn * (\frac{4}{2})^i + n²\] \[ = \ cn* \ \ \sum_{i=0}^{log_2 (n-1)} \ (\frac{4}{2})^i + n²\]

    Pela propriedade do somatório: \( \boxed{\sum_{i=0}^{n} x^{k} \ \ = \ \ \frac{x^{n+1}-1}{x-1}} \)


    \[ = \ cn* \ \ (\frac{(\frac{4}{2})^{lgn}-1}{(\frac{4}{2})-1}) + n²\] \[ = \ cn* \ \ (\frac{\frac{4^{lgn}}{2^{lgn}}-1}{\frac{4-2}{2}}) + n²\] \[ = \ cn* \ \ (\frac{4^{lgn}}{n^{lg2}}-1) + n²\] \[ = \ (\frac{4^{lgn} * c \cancel{n}}{\cancel{n^{1}}}-cn) + n²\] \[ = \ n^{lg4} * c -cn + n²\] \[ = \ 2n² * c -cn\] \[ = \ 2c * n² -cn \ \ <= \ \ 2c * n²\]

    Ou seja, para qualquer constantes c positivas, T(n) = O(n²)







  • Questão 2

    \[\mathbf{T(n) = 4T(\lfloor \frac{n}{2} \rfloor) + cn}\]

    Suponha que T(n) é O(n²), ou seja, T(n) <= c n² para n>=n0, onde c e n0 são constantes positivas.



    Hipótese Indutiva: suponha que para todo k com n0 <= k < n, T(k) <= d k²


    \[T(n) = 4T(\lfloor \frac{n}{2} \rfloor) + cn\] \[ <= \ \ 4 \ (d (\frac{n}{2})^2) + cn\] \[ = \ \ \cancel{4}d \frac{n²}{\cancel{4}} + cn\] \[ = \ \ dn² + cn\]

    Perceba que não conseguimos satisfazer a nossa hipótese. Assim, iremos modificá-la para: T(k) <= d k² - k.


    \[T(n) = 4T(\lfloor \frac{n}{2} \rfloor) + cn\] \[ <= \ \ 4 \ (d (\frac{n}{2})^2 - (\frac{n}{2})) + cn\] \[ = \ \ \cancel{4}d \frac{n²}{\cancel{4}} - \cancel{4}(\frac{n}{\cancel{2}}) + cn\] \[ = \ \ dn² - 2n + cn\] \[ = \ \ dn² - n \boxed{- n + cn}\]

    É necessário mostrar que - n + cn <= 0

    \[ - n + cn \ \ <= \ \ 0\] \[= \ \ n - cn \ \ >= \ \ 0 \ \ \ \ \ \textbf{para n>=0 e c<=1} \]

    1. \(\mathbf{T(n) = 2T(n/4) + 1}\)

      Perceba que podemos identificar as seguintes constantes:

      a=2 b=4 f(n)=1


      Para melhor identificar o caso do teorema mestre, temos que desenvolver \( \ \mathbf{n^{\log_b a}}\)


      \(n^{\log_b a} \ = \ n^{\log_4 2} \ = \ n^{0.5}\)

      Veja que \( \mathbf{f(n) \ = \ 1 \ >= \ n^{0.5 - \varepsilon}}\) \(\ \ \ \ \text{ para } \ \varepsilon \ = \ 0.5\)


      Deste modo, pelo Caso 1 temos que: \(\mathbf{T(n) \ = \ \Theta(n^{0.5})}\)









    2. \(\mathbf{T(n) = 2T(n/4) + n²}\)

      Perceba que podemos identificar as seguintes constantes:

      a=2 b=4 f(n)=n²


      Para melhor identificar o caso do teorema mestre, temos que desenvolver \( \ \mathbf{n^{\log_b a}}\)


      \(n^{\log_b a} \ = \ n^{\log_4 2} \ = \ n^{0.5}\)

      Veja que \( \mathbf{f(n) \ = \ n² \ >= \ n^{0.5 + \varepsilon}}\) \(\ \ \ \ \text{ para } \ \varepsilon \ = \ 0.1\)


      Como estamos tentando provar pelo Caso 3 é necessário mostrar que: \(\mathbf{a \ f(\frac{n}{b}) \ <= \ c \ f(n)}\)


      \[ 2 \ f(\frac{n}{4}) \ <= \ c \ f(n) \] \[ = \ 2 \ (\frac{n}{4})² \ <= \ c \ n² \] \[ = \ \cancel{2} \ \frac{n²}{\cancel{16}} \ <= \ c \ n² \] \[ = \ \frac{n²}{8} \ <= \ c \ n² \ \ \ \ \ \textbf{ para c=0.5}\]

      Logo, pelo Caso 3 temos: \(\mathbf{T(n) \ = \ \Theta(n^{2})}\)

      Logo, pelo Caso 3 temos: \(\mathbf{T(n) \ = \ \Theta(n)}\)
