    1. \(T(n) = T(n/2) + 15, \ \text{onde} \ T(1) = 1.\)

      Suponha que \(T(n) \text{ é } O(\lg n)\). Logo, temos: T(n) <= c lgn para n>=n0, onde c e n0 são constantes positivas.


      Hipótese Indutiva: Suponha que para todo k com n0 <= k < n, T(k) <= c lgk, onde c e n0 são constantes positivas.


      \begin{flalign} T(n) &= T\left( \frac{n}{2}\right) + 15 \\ &\leq c \lg \left(\frac{n}{2}\right) + 15\\ &= c(\lg n - 1) + 15 \\ &= c \lg n - c + 15\\ &\leq c \lg n \end{flalign} para \(c \geq 15\) e qualquer \(n_0\) positivo.

      n=2 basta como caso base.



      Agora temos que provar que \(T(n) \text{ é } \Omega(\lg n)\). Logo, temos: T(n) >= c lgn para n>=n0, onde c e n0 são constantes positivas.


      Hipótese Indutiva: Suponha que para todo k com n0 <= k < n, T(k) >= c lgk, onde c e n0 são constantes positivas.

      \begin{flalign} T(n) &= T\left( \frac{n}{2}\right) + 15 \\ &\geq c \lg \left(\frac{n}{2}\right) + 15\\ &= c(\lg n - 1) + 15 \\ &= c \lg n - c + 15\\ &\geq c \lg n \end{flalign} para \(c \leq 15\) e qualquer \(n_0\) positivo.

      n=1 basta como caso base.






    2. \(F(n) = 2F(n/2) + n, \ \text{onde} \ F(1) = 1.\)

      Suponha que \(T(n) \text{ é } O(n \ lg n)\). Logo, temos: T(n) <= cn lgn para n>=n0, onde c e n0 são constantes positivas.


      Hipótese Indutiva: Suponha que para todo k com n0 <= k < n, T(k) <= ck lgk, onde c e n0 são constantes positivas.

      \begin{flalign} F(n) &= 2F(n/2) + n \\ &\leq \cancel{2} \cdot c \cdot \frac{n}{\cancel{2}} \lg \left( \frac{n}{2} \right) + n\\ &= c \cdot n (\lg n - 1) + n\\ &= c \cdot n \lg n - cn + n \\ &\leq c \lg n \end{flalign} para \(c \geq 1\) e qualquer \(n_0\) positivo.

      n=2 basta como caso base.



      Agora temos que provar que \(T(n) \text{ é } \Omega(\lg n)\). Logo, temos: T(n) >= cn lgn para n>=n0, onde c e n0 são constantes positivas.


      Hipótese Indutiva: Suponha que para todo k com n0 <= k < n, T(k) >= ck lgk, onde c e n0 são constantes positivas.

      \begin{flalign} F(n) &= 2F(n/2) + n \\ &\geq \cancel{2} \cdot c \cdot \frac{n}{\cancel{2}} \lg \left( \frac{n}{2} \right) + n\\ &= c \cdot n (\lg n - 1) + n\\ &= c \cdot n \lg n - cn + n \\ &\geq c \cdot n \lg n \end{flalign} para \(1 \geq c > 0\) e qualquer \(n_0\) positivo.

      n=1 basta como caso base.






    3. \(G(n) = 2G(\sqrt{n}), \ \text{onde} \ G(1) = 1.\)

      Supondo O(n)

      \[ G(n) \leq c n \]

      para algum \( c > 0 \) e para \( n \geq n_0 \), onde \( n_0 \) é um valor base para \( n \).

      Substituindo a hipótese na recorrência:

      \[ G(n) = 2G(\sqrt{n}) \]

      Usando a hipótese de indução:

      \[ G(\sqrt{n}) \leq c \sqrt{n} \]

      Substituindo na recorrência:

      \[ G(n) \leq 2 \cdot c \sqrt{n} \]

      \[ G(n) \leq 2c \sqrt{n} \]

      Para garantir que \( G(n) \leq c n \), precisamos que:

      \[ 2c \sqrt{n} \leq c n \]

      Divida ambos os lados por \( c \sqrt{n} \):

      \[ 2 \leq \sqrt{n} \]

      \[ n \geq 4 \]

      Casos Base

      Para validar a nossa hipótese, verificamos os casos base:

      Para \( n = 4 \):

      \[ G(4) = 2G(\sqrt{4}) = 2G(2) \]

      Se \( n_0 = 4 \), então a hipótese \( G(n) \leq c n \) é válida.

      Portanto, se \( n \geq 4 \), a hipótese de indução está correta para o limite superior. Assim:

      \[ G(n) = O(n) \]

      Limite Inferior

      Hipótese de Indução

      Vamos supor que existe uma constante \( k \) tal que:

      \[ G(n) \geq k n \]

      para algum \( k > 0 \) e para \( n \geq n_0 \), onde \( n_0 \) é um valor base para \( n \).

      Passo Indutivo

      Queremos provar que a hipótese é válida para \( n \) e que:

      \[ G(n) \geq k n \]

      Substituindo a hipótese na recorrência:

      \[ G(n) \geq 2G(\sqrt{n}) \]

      Usando a hipótese de indução:

      \[ G(\sqrt{n}) \geq k \sqrt{n} \]

      Substituindo na recorrência:

      \[ G(n) \geq 2 \cdot k \sqrt{n} \]

      \[ G(n) \geq 2k \sqrt{n} \]

      Para garantir que \( G(n) \geq k n \), precisamos que:

      \[ 2k \sqrt{n} \geq k n \]

      Divida ambos os lados por \( k \sqrt{n} \):

      \[ 2 \geq \sqrt{n} \]

      \[ n \leq 4 \]

      Casos Base

      Para validar a nossa hipótese, verificamos os casos base:

      Para \( n = 4 \):

      \[ G(4) = 2G(\sqrt{4}) = 2G(2) \]

      Se \( n_0 = 4 \), então a hipótese \( G(n) \geq k n \) é válida.

      Portanto, se \( n \leq 4 \), a hipótese de indução está correta para o limite inferior. Assim:

      \[ G(n) = \Omega(n) \]

      Conclusão

      A complexidade assintótica exata da recorrência é:

      \[ G(n) = \Theta(n) \]

    \[ \mathbf{T(n) \ = \ 2T(\lfloor \frac{n}{2} \rfloor \ + \ 17) \ + \ n}\]

    \[ \mathbf{<= \ 2c(\frac{n}{2} \ + \ 17) \ \log_2(\frac{n}{2} \ + \ 17) \ + \ n} \ \ \ \ \ \text{ (A fórmula é maior sem o piso)}\]

    \[ \mathbf{= \ c(n+34) \ (\log_2(n+34) - \log_2(2)) \ + \ n} \ \ \ \ \ \ \ \ \ \ \ \ \text{(Usando a propriedade: } (\log_b(\frac{a}{c}) \ = \ \log_b(a) - \log_b(c))\]
    \[ \mathbf{= \ c(n+34) \ \log_2(n+34) - c(n+34) \ + \ n} \ \ \ \ \ \ \ \ \ \ \ \ \text{(Distribuindo c(n+34). Além disso, lg2 é um) }\]

  • \[ T(n) = 4T\left(\frac{n}{3}\right) + n \]

    Limite Superior

    Hipótese Indutiva: Suponha que para todo k com n0 <= k < n, \( T(k) \leq c k^{\log_3 4} \), onde c e n0 são constantes positivas.

    \[ T(n) = 4T\left(\frac{n}{3}\right) + n \]

    \[ T\left(\frac{n}{3}\right) \leq c \left(\frac{n}{3}\right)^{\log_3 4} \]

    Substituindo na recorrência:

    \[ T(n) \leq 4 \left(c \left(\frac{n}{3}\right)^{\log_3 4}\right) + n \]

    \[ T(n) \leq 4c \left(\frac{n^{\log_3 4}}{3^{\log_3 4}}\right) + n \]

    Como \( 3^{\log_3 4} = 4 \):

    \[ T(n) \leq 4c \left(\frac{n^{\log_3 4}}{4}\right) + n \]

    \[ T(n) \leq c n^{\log_3 4} + n \]

    Para garantir que \( T(n) \leq c n^{\log_3 4} \), precisamos que:

    \[ c n^{\log_3 4} + n \leq c n^{\log_3 4} \]

    \[ n \leq 0 \]

    Essa desigualdade nunca é verdadeira para \( n > 0 \), então a hipótese falha.

    Hipótese Ajustada

    \[ T(n) \leq c n^{\log_3 4} - k n \ \ \ \text{ (onde \( k \) é uma constante positiva).}\]

    Substituindo na recorrência:

    \[ T(n) \leq 4 \left(c \left(\frac{n}{3}\right)^{\log_3 4} - k \frac{n}{3}\right) + n \]

    \[ T(n) \leq 4 \left(c \frac{n^{\log_3 4}}{4} - k \frac{n}{3}\right) + n \]

    \[ T(n) \leq c n^{\log_3 4} - \frac{4k n}{3} + n \]

    Para garantir que \( T(n) \leq c n^{\log_3 4} - k n \), precisamos que:

    \[ - \frac{4k n}{3} + n \leq -k n \]

    \[ n - \frac{4k n}{3} \leq -k n \]

    \[ n \leq \frac{4k n}{3} - k n \]

    \[ n \leq \frac{k n}{3} \]

    \[ 1 \leq \frac{k}{3} \]

    Multiplique por *3

    \[ 3 \leq K \]

    Com \( k \) suficientemente grande, \( k \geq 3 \), a hipótese é válida.
