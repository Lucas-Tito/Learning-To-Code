A ultima linha eu deduzi que seria log n pois o n sempre está sendo dividido pela metade. Algo como n/2, n/4, /n8...1, com isso o algoritimo seria \( O(N \log N) \).

A recorrencia completa seria \( T(N) = N + T\left(\left\lfloor \frac{N}{2} \right\rfloor \right) \).

Vou utilizar metódo de substituição para provar que a complexidade é \( O(N \log N) \) assim, usando indução em N. Primeiramente, a hipótese de indução:

Hipótese de indução: Para todo inteiro K < N, T(K) <= C * K Log K. sendo N0 < K < N.

Passo indutivo: Considere uma entrada de tamanho N suficientemente grande. Quero provar que \(T(N) <= C * N \log N\)

Substituindo isso na relação de recorrência:

T(N) = N + T(N / 2)
T(N) ≤ N + c (N / 2) log (N / 2)

Usamos a propriedade log (N / 2) = log N - log 2:

T(N) ≤ N + c (N / 2) (log N - log 2)
T(N) ≤ N + c (N / 2) log N - c (N / 2) log 2
T(N) ≤ N + c * N log N - c (N / 2) log 2

Agora preciso retirar a parte negativa então vou usar \( N - \frac{cN}{2} \log 2 < 1 \) com o N excedente.

Prova da Desigualdade

Para provar a desigualdade:

\( N - \frac{cN}{2} \log 2 < 1 \)

1. Reescrever a Inequação

Reescrevemos a inequação para isolar \( c \):

\( N - \frac{cN}{2} \log 2 < 1 \)

Subtraímos \( N \) de ambos os lados:

\( - \frac{cN}{2} \log 2 < 1 - N \)

Multiplicamos ambos os lados por \(-1\) (invertendo a desigualdade):

\( \frac{cN}{2} \log 2 > N - 1 \)

Dividimos ambos os lados por \( \frac{N}{2} \log 2 \):

\( c > \frac{2(N - 1)}{N \log 2} \)

2. Escolher \( c \) Adequado

Para garantir que a desigualdade seja verdadeira, escolhemos \( c \) maior que:

\( c > \frac{2(N - 1)}{N \log 2} \)

3. Exemplo Numérico

Se \( \log 2 \approx 0.693 \), então:

\( \frac{2}{\log 2} \approx \frac{2}{0.693} \approx 2.89 \)

Então, para garantir que a desigualdade seja verdadeira, escolhemos \( c \geq 3 \).

Escolhendo \( c \geq 3 \), garantimos que:

\( N - \frac{cN}{2} \log 2 < 1 \)

para todos os \( N \) suficientemente grandes.

Conclusão: Com isso, consigo provar que a complexidade é \( T(N) ≤ N \log N\)

Prova dos Casos Base para o Algoritmo

A função de recorrência para o tempo de execução do algoritmo é:

\( T(N) = N + T\left(\left\lfloor \frac{N}{2} \right\rfloor\right) \)

Caso Base: \( T(1) \)

Para \( N = 1 \), o algoritmo faz o seguinte:

  • Calcula a soma dos números de 1 até 1, que é \( 1 \).
  • Chama a função recursivamente com \( N/2 \), onde \( N/2 = 0 \).

Portanto:

\( T(1) = 1 + T(0) \)

De acordo com o caso base definido, temos:

\( T(0) = 0 \)

Assim:

\( T(1) = 1 + 0 = 1 \)

Verificação da Hipótese para \( T(1) \)

Suponha que nossa hipótese é que a complexidade de tempo de execução \( T(N) \) é \( O(N \log N) \). Verificamos a hipótese para \( N = 1 \):

\( T(1) \leq c \cdot 1 \cdot \log 1 \)

Sabemos que \( \log 1 = 0 \), então:

\( T(1) \leq c \cdot 1 \cdot 0 = 0 \)

Portanto, precisamos que:

\( 1 \leq 0 \)

Esta desigualdade não é verdadeira. Portanto, a hipótese \( O(N \log N) \) não é válida para \( N = 1 \). Em vez disso, a complexidade correta para \( N = 1 \) é \( O(1) \).

Caso Base: \( T(2) \)

Para \( N = 2 \), o algoritmo faz o seguinte:

  • Calcula a soma dos números de 1 até 2, que é \( 1 + 2 = 3 \).
  • Chama a função recursivamente com \( N/2 \), onde \( N/2 = 1 \).

Portanto:

\( T(2) = 2 + T(1) \)

Como já sabemos que:

\( T(1) = 1 \)

Temos:

\( T(2) = 2 + 1 = 3 \)

Verificação da Hipótese para \( T(2) \) com \( c = 3 \)

Nossa hipótese é que a complexidade de tempo de execução \( T(N) \) é \( O(N \log N) \). Verificamos para \( N = 2 \) com \( c = 3 \):

\( T(2) \leq c \cdot 2 \cdot \log 2 \)

Como \( \log 2 = 1 \), então:

\( T(2) \leq 3 \cdot 2 \cdot 1 = 6 \)

Como \( T(2) = 3 \), verificamos:

\( 3 \leq 6 \)

A desigualdade é verdadeira. Portanto, a hipótese \( O(N \log N) \) é válida para \( c = 3 \) e \( N = 2 \).

Conclusão

A hipótese é aceita a partir de N = 2.

A recorrência dada é:

Limite Superior

Hipótese de Indução

Vamos supor que existe uma constante \( c \) tal que:

\[ T(n) \leq c n^{\log_3 4} \]

para algum \( c > 0 \) e para \( n \geq n_0 \), onde \( n_0 \) é um valor base para \( n \).

Passo Indutivo

Queremos provar que a hipótese é válida para \( n \) e que:

\[ T(n) \leq c n^{\log_3 4} \]

Substituindo a hipótese na recorrência:

\[ T(n) = 4T\left(\frac{n}{3}\right) + n \]

Usando a hipótese de indução:

\[ T\left(\frac{n}{3}\right) \leq c \left(\frac{n}{3}\right)^{\log_3 4} \]

Substituindo na recorrência:

\[ T(n) \leq 4 \left(c \left(\frac{n}{3}\right)^{\log_3 4}\right) + n \]

\[ T(n) \leq 4c \left(\frac{n^{\log_3 4}}{3^{\log_3 4}}\right) + n \]

Como \( 3^{\log_3 4} = 4 \):

\[ T(n) \leq 4c \left(\frac{n^{\log_3 4}}{4}\right) + n \]

\[ T(n) \leq c n^{\log_3 4} + n \]

Para garantir que \( T(n) \leq c n^{\log_3 4} \), precisamos que:

\[ c n^{\log_3 4} + n \leq c n^{\log_3 4} \]

\[ n \leq 0 \]

Essa desigualdade nunca é verdadeira para \( n > 0 \), então a hipótese falha.

Hipótese Ajustada

Vamos ajustar a hipótese para:

\[ T(n) \leq c n^{\log_3 4} - k n \]

onde \( k \) é uma constante positiva.

Substituindo na recorrência:

\[ T(n) \leq 4 \left(c \left(\frac{n}{3}\right)^{\log_3 4} - k \frac{n}{3}\right) + n \]

\[ T(n) \leq 4 \left(c \frac{n^{\log_3 4}}{4} - k \frac{n}{3}\right) + n \]

\[ T(n) \leq c n^{\log_3 4} - \frac{4k n}{3} + n \]

Para garantir que \( T(n) \leq c n^{\log_3 4} - k n \), precisamos que:

\[ - \frac{4k n}{3} + n \leq -k n \]

\[ n - \frac{4k n}{3} \leq -k n \]

\[ n \leq \frac{4k n}{3} - k n \]

\[ n \leq \frac{k n}{3} \]

\[ 1 \leq \frac{k}{3} \]

Multiplique por *3

\[ 3 \leq K \]

Portanto, se escolhermos \( k \) suficientemente grande, por exemplo \( k \geq 3 \), a hipótese ajustada é válida.

Conclusão

A complexidade assintótica superior é:

\[ T(n) = O(n^{\log_3 4}) \]
