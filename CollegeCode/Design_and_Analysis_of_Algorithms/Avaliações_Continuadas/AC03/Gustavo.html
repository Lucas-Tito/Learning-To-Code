  • Questão 1. Utilize o método da substituição para calcular a complexidade de tempo de pior caso do algoritmo abaixo. No algoritmo abaixo, o parâmetro \(N\) da função é um número inteiro. A operação de divisão usada é a divisão inteira, ou seja, é a divisão que sempre resulta em um inteiro. 

    
        Algoritmo(N):
          Se N == 0 entao
            Retorne 0
          Senao
            soma = 0
            Para j = 1, ..., N faça
              soma = soma + j
            Retorne soma + Algoritmo(N/2)

    • Solução:

      \[ \begin{array}{|l|l|l|} \hline \text{Algoritmo(N)} & \text{Custo} & \text{Qntd. vezes} \\ \hline \text{1. Se N == 0 entao retorna 0} & c1 & 1 \\ \hline \text{2. Senao} & c2 & 1 \\ \hline \text{3.} \quad \text{ Para j = 1, ..., N faça} & c3 & N + 1 \\ \hline \text{4.} \quad \quad \text{soma = soma + j} & c4 & N \\ \hline \text{5. Retorne soma + Algoritmo(N/2)} & c5 & T(\lfloor N/2 \rfloor) \\ \hline \end{array} \]

      Desse modo, verificamos que \(T(N) = N + T(\lfloor N/2 \rfloor)\)

      Queremos provar que \( T(N) = O(N*log_2(N)) \)

      Ou seja, \(T(N) \leq c*N*log_2(N)\)

      Vamos provar por indução em N:

      Provando caso base N = 2:

      Para \(n_0\) = 2,

      \( T(2) = 2 + T(1) = 2 + 2 + T(0) = 4 + T(0) = 4 \)

      \(\leq 4*log_2(2) \leq 4*log_2(2) = 4*1\)

      Como \(4 \leq 4, n_0 = 2\) pode ser considerado o caso base


      Hipótese de indução: Suponha que \( T(k) \leq c *k*log_2(k), \forall n_0 \leq k < n \)

      Passo indutivo: Queremos provar \(T(N) = c*N*log(N)\)

      \( T(N) = N + T(\lfloor N/2 \rfloor) \)

      Substituição:

      \( T(N) = N + c * (\lfloor N/2 \rfloor) * log_2(\lfloor N/2 \rfloor)) \leq T(N) = N + c * (N/2) * log_2(N/2) \leq N + c *(N/2) * (log_2(N) - 1)\)
      \(= N + c*(N/2)*log_2(N) - c*(N/2)\)

      Agora precisamos provar que \(N - c*(N/2) \leq 0\)

      Se considerarmos c = 2, teriamos:

      \( N - 2*(N/2) = N - N = 0 \)

      Logo, provamos que \(N - c*(N/2) \leq 0\)

      Portanto, \(c * (N/2) * log_2(N) - c * (N/2) \leq c * (N/2) * log_2(N)\)

      e \(c * (N/2) * log_2(N) \leq c * (N) * log_2(N)\) (como queriamos comprovar)


  • Questão 2. Utilizando o método de substituição, prove limitantes superiores e inferiores assintóticos para as recorrências a seguir:

    1. \(T(n) = T(n/2) + 15, \ \text{onde} \ T(1) = 1.\)

      • Solução:

        Queremos provar que \( T(n) = \Theta(lg(n)) \)

        Pela definição de \( \Theta\), \( \exists c_1, c_2, n_0 | 0 \leq c_1*lg(n) \leq T(n) \leq c_2*lg(n) \)

        Caso base: \(n = 2\)

        Parte 1: \( T(2) \geq c_1*lg(2) \)

        \( T(2) = T(1) + 15 = 1 + 15 = 16 \leq c_1*lg(2) = c_1\), que funciona para \( c_1 \geq 16\)

        , logo, o caso base n = 2 funciona para a parte 1

        Parte 2: \( T(2) \leq c_2*lg(2) \)

        \( T(2) = T(1) + 15 = 16 \leq c_1*lg(2) = c_1 \), que funciona para \(c_1 = 16\)

        Vamos provar por indução em n

        Hipótese de indução: \( 0 \leq c_1*lg(k) \leq T(k) \leq c_2*lg(k), \forall n_0 \leq k < n \)

        Parte 1: \( T(n) \geq c_1*lg(n)\)

        \( T(n) = T( \frac{n}{2} ) + 15 \)

        Substituição:

        \( T(n) \geq c_1*lg( \frac{n}{2} ) + 15 \)

        \( = c_1*(lg(n) - 1) + 15 \)

        \( = c_1*lg(n) - c_1 + 15 \geq c_1*lg(n), \) para \( c_1 = 15 \) e n > 1

        Parte 2: \( T(n) \leq c_2*lg(n) \)

        \( T(n) = T( \frac{n}{2} ) + 15 \)

        Substituição:

        \( T(n) \leq c_2*lg( \frac{n}{2} ) + 15 \)

        \( = c_2*(lg( n ) - 1) + 15 \)

        \( = c_2*lg( n ) - c_2 + 15 \leq c_2*lg(n),\) para \( c_2 \geq 15\) e n > 1

        Como provamos tanto para a parte 1 quanto para a parte 2, então \( T(n) = \Theta(lg(n)) \)

    2. \(F(n) = 2F(n/2) + n, \ \text{onde} \ F(1) = 1.\)
      • Solução:

        Queremos provar que \( F(n) = \Theta(n*lg(n)) \)

        Pela definição de \( \Theta\), \( \exists c_1, c_2, n_0 | 0 \leq c_1*n*lg(n) \leq F(n) \leq c_2*n*lg(n) \)

        Caso base: n = 2

        Parte 1: \( F(2) = 2*F( \frac{2}{2}) + 2 = 2 + 2 = 4 \geq c_1*2*lg(2) = c_1*2\), para \( c_1 = 2 \)

        Parte 2: \( F(2) = 2*F( \frac{2}{2}) + 2 = 2 + 2 = 4 \leq c_2*2*lg(2) = c_2*2\), para \( c_2 \geq 2 \)

        Hipótese de indução: \( 0 \leq c_1*n*lg(k) \leq F(k) \leq c_2*n*lg(k) \)

        Parte 1: \( F(n) \geq c_1*n*lg(n) \)

        \( F(n) = 2*F( \frac{n}{2}) + 2 \)

        Substituição:

        \( F(n) \geq \cancel{2}*c_1*( \frac{n}{\cancel{2}} )*lg( \frac{n}{2} ) + n = c_1*n*(lg(n) - 1) + n\)

        \( = c_1*n*lg(n) - c_1*n + n \geq c_1*n*lg(n) \), para \(c_1 \geq 1 \) e n_0 positivo

        Parte 2: \( F(n) \leq c_2*n*lg(n) \)

        \( F(n) = 2*F( \frac{n}{2}) + 2 \)

        Substituição:

        \( F(n) \leq \cancel{2}*c_2*( \frac{n}{\cancel{2}} )*lg( \frac{n}{2} ) + n = c_2*n*(lg(n) - 1) + n\)

        \( = c_2*n*lg(n) - c_2*n + n \leq c_2*n*lg(n) \), para \(c_2 = 1 \) e n_0 positivo

        Como provamos tanto para a parte 1 quanto para a parte 2, então \( F(n) = \Theta(n*lg(n))\)

    3. \(G(n) = 2G(\sqrt{n}), \ \text{onde} \ G(1) = 1.\)
      • Solução:

        Limite superior:

        Queremos provar que G(n) = O(n)

        Pela definição de \( O \), temos que \( \exists c, n_0 | 0 \leq G(n) \leq c_2*(n) \)

        Caso base: \( n = 4 \)

        \( G(4) = 2*c*\sqrt{4} \) \( = 2*c*2 = 4*c \leq 4 \), para \( c \geq 1 \)

        Hipótese de indução: \( 0 \leq G(k) \leq c*lg(k), \forall n_0 \leq k < n \)

        \( G(n) = 2*G( \sqrt(n) ) \)

        Substituição:

        \( G(n) \leq 2*c*n*lg( \sqrt{n} ) = \cancel{2}*c*n \frac{1}{\cancel{2}}*lg(n) \)

        \( = c*n*lg(n) \leq c*n*lg(n) \), para \( c \geq 1\) e \( n\) positivo, como queriamos comprovar.

        Limite superior:

        Queremos provar que G(n) = \( \Omega(lg(n)) \)

        Pela definição de \( \Omega \), temos que \( \exists c, n_0 | 0 \leq c_2*(n) \leq G(n) \)

        Caso base \(n = 4\)

        \( G(4) = 2*c*lg(\sqrt{4}) \) \( = \cancel{2}*c* \frac{1}{ \cancel{2}} lg(n) \)
        \( = c*lg(n) \leq c*lg(n) \), para \( n \geq 1 \) e \( c \geq 1 \).

        Hipótese de indução: \( 0 \leq c*lg(k) \leq G(k), \forall n_0 \leq k < n \)

        \( G(n) = 2*G(\sqrt{n}) \)

        Substituição:

        \( G(n) \leq 2*c*lg( \sqrt{n} ) \)

        \( = \cancel{2} *c* \frac{1}{\cancel{2}} * lg(n) \)

        \( = c*lg(n) \leq c*lg(n) \), para \( c \geq 1\) e \( n\) positivo, como queriamos comprovar.

  • Questão 3. Usando o método da substituição, prove que a solução para a relação de recorrência \(T(n) = 2T(\lfloor n/2 \rfloor + 17) + n\) é \(O(n \lg n)\).

    • Solução:

      Queremos provar nem \(T(n) = 2*T(\lfloor n/2 + 17\rfloor) + n\) é \( O(n*lg*n) \)

      * Pela def. de Big O: \( \exists c, n_0 | T(n) \leq c * n * lg (n), \forall n >= n_0 \)

      Vamos provar a indução em n

      Hipótese de indução: \(T(k) \leq c * k * lg(k) \)

      Passo indutivo: Queremos provar que \(T(n) \leq c * n * lg(n)\)

      \(T(n) = 2*T(\lfloor n/2 \rfloor + 17) + n \)

      Fazendo a substituição, temos:

      \( T(n) \leq 2*c*(\lfloor n/2 \rfloor + 17)*lg(\lfloor n/2 \rfloor + 17) + n \leq 2*c*(n/2 + 17)*lg(n/2 + 17) + n \)

      \(= \cancel{2} * c * ( \frac{n + 34}{\cancel{2}})*lg( \frac{n + 34}{2} ) + n \)

      \(= c*(n + 34)*lg(n + 34) - c*(n + 34) + n \)

      Como não foi possível chegarmos ao formato esperado, a hipótese deve ser alterada

      Hipótese de indução 2: \(T(k) \leq c*(k-17)*lg(k - 17)\)

      Passo indutivo: Queremos provar que \(T(n) \leq c*(n - 17)*lg(n-17) \)

      \( T(n) = 2* c * ( \lfloor n/2 \rfloor + 17 - 17) * lg( \lfloor n/2 \rfloor + 17 - 17) + n \)

      \( \leq 2* c * ( n/2 ) * lg( n/2 ) + n = \cancel{2} * c * ( \frac{n}{\cancel{2}} ) * lg( \frac{n}{2} ) + n \)

      \( c * (n) * lg(\frac{n}{2}) + n = c * (n) * (lg(n) - 1) + n = c*n*lg(n) - c*n + n\)

      Precisamos provar que \( -c*n + n \leq 0\), isso é verdade para \( c \geq 1 \) e \( n \geq 1 \)

      Portanto, temos que: \(c*n*lg(n) - c*n + n \leq c*n*lg(n) \). Como queriamos comprovar.

  • Questão 4. É possível provar que a solução para a recorrência \(T(n) = 4T(n/3)+ n\) é \(T(n) = \Theta(n^{\log_3 4})\). Mostre que uma prova de substituição que considere \(T(n) \leq cn^{\log_3 4}\) falha. Depois, mostre como subtrair um termo de ordem mais baixa para fazer com que uma prova de substituição funcione. Basta provar apenas o limitante superior, não precisa provar o inferior.

    • Solução:

      Vamos provar por absurdo que \( T(n) \leq c*n^{log_3{4}} \) falha

      Para isso, vamos supor que \( T(n) \leq c*n^{log_3{4}} \) é verdade

      \( T(n) = 4*T( \frac{n}{3} ) + n \)

      Aplicando a substituição:

      \( T(n) \leq 4*c*( \frac{n}{3} )^{log_3{4}} + n \) = \( 4*c*( \frac{n^{log_3{4}}}{\cancel{3}^{log_\cancel{3}{4}}} ) + n \)

      \( \cancel{4}*c*( \frac{n^{log_3{4}}}{\cancel{4}} ) + n \) = \( c*n^{log_3{4}} + n \)

      Desse modo, para que fosse verdade, n deveria ser menor ou igual a zero (Absurdo)
