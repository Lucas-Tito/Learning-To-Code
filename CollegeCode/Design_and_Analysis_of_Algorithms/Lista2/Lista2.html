    \[ \begin{array}{|l|l|l|} \hline \text{Algoritmo(S, X)} & \text{Custo} & \text{Qntd. vezes} \\ \hline \text{1. Para i de 1 até S faça} & c1 & N \\ \hline \quad\text{2. Para j de i+1 até S faça} & c2 & \sum_{i=1}^{N-1} (N-i+1) \\ \hline \qquad\text{3. Se S[i] + S[j] = X} & c3 & \sum_{i=1}^{N-1} (N-i) \\ \hline \qquad \quad \text{4. Retorne V} & c4 & 1 \\ \hline \text{5. Retorne F} & c5 & 1 \\ \hline \end{array} \]
    Comentários:

    Linha 2: O for da linha 2 executa essa quantidade de vezes já que o primeiro for vai até N, mas a cada execução do primeiro para, o segundo for roda mais N vezes.

    Além disso, o +1 é por conta do incremento de falha.


    \[ \mathbf{\text{Melhor e Pior Caso}}\]

    No melhor dos casos S[i]+S[j] = X será verdadeiro na primeira execução do primeiro e do segundo for. Assim, as linhas 1, 2 e 3 serão executadas apenas uma vez.

    Ou seja, O(1)


    No pior caso, S[i]+S[j] = X nunca é satisfeito, o que obriga todas as linhas serem executadas ao máximo.

    Desse modo, começaremos desenvolvendo o somatório da linha 2:

    \[ \sum_{i=1}^{N-1} (N-i+1) \ = \ \sum_{i=1}^{N-1} (N) \ - \ \sum_{i=1}^{N-1} (i) \ + \ \sum_{i=1}^{N-1} (1) \] \[ = \ N \sum_{i=1}^{N-1} (1) \ - \ \frac{N(N-1)}{2} \ + \ N-1 \] \[ = \ N(N-1) \ - \ \frac{N² - N}{2} \ + \ N-1\] \[ = \ N² - N \ - \ \frac{N² - N \ + \ 2N - 2}{2} \] \[ = \ \frac{2N² - 2N \ - \ N² - N \ + \ 2N - 2}{2} \] \[ = \ \mathbf{\frac{N² - N - 2}{2}}\]

    Agora iremos desenvolver o somatório da linha 3:

    \[ \sum_{i=1}^{N-1} (N-i) \ = \ \sum_{i=1}^{N-1} (N) \ - \ \sum_{i=1}^{N-1} (i) \] \[ = \ N \sum_{i=1}^{N-1} (1) \ - \ \frac{N(N-1)}{2}\] \[ = \ N(N-1) \ - \ \frac{N² - N}{2} \] \[ = \ N²-N \ - \ \frac{N² - N}{2} \] \[ = \frac{2N²-2N \ - \ N² - N}{2} \] \[ = \mathbf{\frac{N²-N}{2}} \]

    Agora que temos os dois somatórios, iniciaremos o cálculo dos custos:

    \[ T(N) \ = \ c1*N \ \ + \ \ c2*(\frac{N²-N-2}{2}) \ \ + \ \ c3*(\frac{N²-N}{2}) \ \ + \ \ c4 + c5\]

    O que configura uma função da forma T(N) = an²+bn+c, onde a, b e c são constantes.

    Portanto, O(n²).

    Por definição uma função limitada por O(g(n)) significa que f(n) <= c*g(n) para todo n>= n₀, logo temos:

    \[ \mathbf{2 \log_2 (n) \ \ >= \ \ c* \log_2 (n+5)} \]

    \[ \mathbf{2n \ \ >= \ \ n+5} \] \[ \mathbf{2*6 \ \ >= \ \ 6+5 \ \ \ \ = \ \ \ \ 12 \ \ >= \ \ 11} \]
